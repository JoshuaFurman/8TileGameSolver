This project solves the 8-tile puzzle game in the least amount of moves possible using the A* graph traversal algorithm.

To use this project:
- Make sure you have pipenv installed (i.e. pipenv --version)
- To activate the virtual environment run the command: pipenv shell
- To begin run the command: python main.py
- Click on a cell to fill in a number (0 represents the empty tile space)
- If the configuration is solvable you will be prompted to press the space bar to begin solving the puzzle.
